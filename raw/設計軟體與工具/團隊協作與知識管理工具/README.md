# 團隊協作與知識管理工具

## 軟體分類

| 軟體 | 說明 |
|---|---|
| [Notion](Notion/) | 專案知識庫、任務、會議紀錄、資料庫與 SOP 管理 |
| [Teams](Teams/) | 會議、聊天、檔案、頻道與 Microsoft 365 協作 |
| [Slack](Slack/) | 團隊訊息、頻道、整合、自動化與快速協作 |
| [Google Drive](GoogleDrive/) | 雲端檔案、共享、協作文件與版本管理 |
| [OneDrive](OneDrive/) | Microsoft 雲端檔案、同步、共享與 Office 協作 |
| [Synology Drive](SynologyDrive/) | 私有雲檔案同步、權限、版本與事務所內部資料管理 |
