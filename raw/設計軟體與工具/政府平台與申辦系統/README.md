# 政府平台與申辦系統

## 平台分類

| 平台 | 說明 |
|---|---|
| [政府電子採購網](政府電子採購網/) | 標案查詢、招標公告、決標公告、電子領標與投標文件流程 |
| [全國建築管理資訊系統](全國建築管理資訊系統/) | 建管資訊、建築師開業、建照使照查詢與公共安全申報相關入口 |
| [地方政府建照申辦系統](地方政府建照申辦系統/) | 各縣市建照線上申辦、書圖文件繳交、勘驗與補正流程 |
| [都市計畫圖資查詢系統](都市計畫圖資查詢系統/) | 都市計畫圖、使用分區、土地管制與計畫書圖查詢 |
| [國土測繪圖資服務雲](國土測繪圖資服務雲/) | 地籍圖、正射影像、通用電子地圖與國土測繪圖層套疊 |
| [建築法規查詢系統](建築法規查詢系統/) | 建築技術規則、建築相關法規與解釋函令查詢 |
| [公共工程規範與工程會系統](公共工程規範與工程會系統/) | 公共工程施工綱要規範、採購法規、工程會資料與公共工程文件 |
