# 渲染與即時視覺化工具

## 軟體分類

| 軟體 | 說明 |
|---|---|
| [Enscape](Enscape/) | 即時渲染、VR、BIM 模型快速視覺化與設計溝通 |
| [Twinmotion](Twinmotion/) | 即時場景、植栽、人車、動畫與建築展示 |
| [Lumion](Lumion/) | 建築動畫、景觀氛圍、快速外觀視覺化與影片輸出 |
| [V-Ray](VRay/) | 高品質渲染、材質、燈光、相機與專業視覺化 |
| [Corona](Corona/) | 建築室內外高品質渲染、材質與光影表現 |
| [D5 Render](D5Render/) | 即時渲染、資產庫、動畫與快速建築視覺化 |
