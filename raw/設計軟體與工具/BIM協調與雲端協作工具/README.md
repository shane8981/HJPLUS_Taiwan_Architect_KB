# BIM協調與雲端協作工具

## 軟體分類

| 軟體 | 說明 |
|---|---|
| [Navisworks](Navisworks/) | 模型整合、碰撞檢查、4D 模擬與協調會議 |
| [Solibri](Solibri/) | IFC 模型審查、規則檢核、分類與資訊品質檢查 |
| [BIMcollab](BIMcollab/) | BCF 議題管理、模型協調、碰撞追蹤與團隊協作 |
| [Autodesk Construction Cloud](AutodeskConstructionCloud/) | 雲端圖面、模型、文件、審查與施工協作平台 |
| [BIM 360](BIM360/) | Autodesk 舊版雲端 BIM 協作、Docs、Coordinate 與 Field 流程 |
