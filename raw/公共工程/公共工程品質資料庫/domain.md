# 公共工程品質資料庫

## 使用情境

這個知識入口整理台灣公共工程中與施工品質、監造查證、交通維持、職業安全衛生、材料設備檢試驗、施工查驗標準與施工綱要規範相關的公開來源。適合在下列情境使用：

- 查找公共工程施工品質管理作業要點、監造與品質計畫綱要、材料設備檢試驗表單。
- 比對施工品質、監造、交通維持與職業安全衛生資料是否引用到正確來源。
- 尋找自主檢查表、施工抽查標準、抽查紀錄表與品質管理標準表格。
- 查找工程會 PCIC 施工綱要規範、地方政府施工規範與道路工程品質查驗資料。
- 整理正式送審前的資料來源清單與法規查核清單。

## 學習目標

使用者應能掌握：

1. 公共工程施工品質管理常用官方來源與地方政府公開範本。
2. 品質文件、監造文件、材料檢試驗、施工查驗與規範章節之間的對應關係。
3. 舊版範本與現行法規之間的差異風險。
4. 正式文件送審前需要回到契約、圖說、主辦機關要求與最新官方法規校核。

## 資料範圍

| 類別 | 內容 |
| --- | --- |
| 官方綱要與 SOP | 工程會監造計畫暨品質計畫製作綱要、地方政府品質管理 SOP |
| 施工品質與監造 | 施工品質計畫、監造計畫、低金額格式、地方政府參考格式 |
| 交通維持 | 交通維持計畫制式範本、道路活動格式與縣市案例 |
| 職業安全衛生 | 職安衛管理計畫、安全衛生稽查表、危險性作業表單 |
| 表單與檢核 | 材料設備送審、檢試驗管制、施工日誌、監造報表與審查表 |
| 施工查驗標準 | 品質管理標準、自主檢查表、施工抽查標準與抽查紀錄表 |
| 施工規範與技術規則 | 工程會 PCIC 施工綱要規範、地方施工規範、道路設計與技術標準 |

## 主要官方來源

- [行政院公共工程委員會](https://www.pcc.gov.tw/)
- [公共工程施工品質管理作業要點](https://lawweb.pcc.gov.tw/LawContent.aspx?id=FL003734)
- [工程會 PCIC 公共工程施工綱要規範](https://pcic.pcc.gov.tw/)
- [臺北市政府採購業務資訊網施工品質安全下載專區](https://gpis.taipei/RWD/frontfunction/Quality/wfrmDownload.aspx?print=1)
- [勞動部職業安全衛生署](https://www.osha.gov.tw/)

## 實務提醒

- 公開範本可作架構與欄位參考，但正式送審仍需依個案契約、圖說、特訂規範、主辦機關規定與地方審查要求校核。
- 施工查驗表格應對應實際工程項目與契約數量，不宜只套用通用章節名稱。
- 交通維持與職業安全衛生要求高度依地方主管機關與工地條件而異，跨縣市使用前需確認當地格式。
- 若資料庫以壓縮檔形式分享，建議放在 GitHub Releases 或其他下載空間，不要把大型 zip 直接放入 git repository。
