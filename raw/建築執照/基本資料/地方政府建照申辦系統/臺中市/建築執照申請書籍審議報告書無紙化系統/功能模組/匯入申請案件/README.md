# 匯入申請案件

臺中市二維系統提供線上版案件匯入與單機版案件匯入。

## 匯入類型

| 類型 | 說明 | 操作手冊 |
|------|------|----------|
| [線上版案件匯入](線上版案件匯入/) | 輸入案件匯出時產生的 32 位數代碼後送出。 | https://mcgbm.taichung.gov.tw/arch/pdf/case_import_web.pdf |
| [單機版案件匯入](單機版案件匯入/) | 選擇案件類別、發照單位，視需要輸入一碼通，並夾帶單機版系統產出的 data.txt。 | https://mcgbm.taichung.gov.tw/arch/pdf/case_import_single.pdf |
