# 人員授權管理

建築師帳號或營造業主帳號可在人員授權管理頁面授權員工使用系統。

## 已知規則
- 開業建築師需使用自然人憑證登入，不可使用團體憑證。
- 營造廠端須使用工商憑證登入。
- 員工第一次登入後會是未授權狀態。
- 建築師或營造業主帳號只需員工的「帳號」與「姓名」即可授權。
- 被授權員工可決定是否開啟管理權限。
- 有管理權限的員工可協助授權其他人員，並可管理所有案件。
- 無管理權限的一般員工只可看見被授權的案件。
- 跑照等特殊身分可被多家事務所或營造業授權。
