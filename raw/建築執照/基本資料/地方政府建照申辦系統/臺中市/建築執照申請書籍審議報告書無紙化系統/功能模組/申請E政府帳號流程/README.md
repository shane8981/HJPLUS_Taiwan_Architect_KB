# 申請E政府帳號流程

事務所與專業技師使用自然人憑證或是工商憑證登入
一般人員需先申請 E 政府帳號，才能登入臺中市二維系統中對應的事務所或營造業系統。

## 已知規則
- 一般人員帳號掛在建築師或營造業帳號底下。
- 第一次登入後，員工端預設為未授權狀態。
- E 政府帳號申請網址：https://www.cp.gov.tw/portal/cpinit/RegisterM.aspx?ReturnUrl=https%3a%2f%2fmcgbm.taichung.gov.tw%2fegov%2farch%2floginok
