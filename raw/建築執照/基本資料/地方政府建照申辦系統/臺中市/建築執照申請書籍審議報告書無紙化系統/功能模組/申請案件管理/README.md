# 申請案件管理

申請案件管理是建築執業人員最常使用的臺中市二維系統功能。使用者登入後，若被多家事務所或營造業授權，需先在首頁切換使用的事務所或營造業資料，再從左上角漢堡選單開啟申請案件管理。

## 子功能

| 子功能 | 說明 |
|--------|------|
| [新增案件](新增案件/) | 預留新增案件流程。 |
| [建造執照](建造執照/) | 以 Web 版申請或管理建造執照案件。 |
| [使用執照](使用執照/) | 以 Web 版申請或管理使用執照案件。 |
| [拆除執照](拆除執照/) | 以 Web 版申請或管理拆除執照案件。 |
| [變更設計](變更設計/) | 以 Web 版申請或管理變更設計案件。 |
| [預審申請](預審申請/) | 以 Web 版申請或管理預審案件。 |
| [室內裝修申請](室內裝修申請/) | 以 Web 版申請或管理室內裝修案件。 |
