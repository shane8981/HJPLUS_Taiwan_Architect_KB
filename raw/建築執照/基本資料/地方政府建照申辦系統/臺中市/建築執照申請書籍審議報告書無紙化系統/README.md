# 建築執照申請書籍審議報告書無紙化系統

本條目整理臺中市 Web 版建築執照申請書籍審議報告書無紙化系統，也就是實務上常稱的「二維系統」。

## 核心文件

| 文件 | 說明 |
|------|------|
| [domain.md](domain.md) | 人類閱讀用的臺中市二維系統操作脈絡與實務說明。 |
| [taichung-paperless-building-permit-documents/SKILL.md](taichung-paperless-building-permit-documents/SKILL.md) | AI 使用的系統判斷、權限模型、案件管理與匯入規則。 |

## 功能模組

| 模組 | 說明 |
|------|------|
| [申請E政府帳號流程](功能模組/申請E政府帳號流程/) | 事務所、營造業一般人員登入前需先申請 E 政府帳號。 |
| [人員授權管理](功能模組/人員授權管理/) | 建築師或營造業主帳號授權員工，並可設定管理權限。 |
| [申請案件管理](功能模組/申請案件管理/) | 建築執業人員最常使用的案件新增、查詢、管理與授權案件作業。 |
| [匯入申請案件](功能模組/匯入申請案件/) | 線上版案件匯入與單機版案件匯入。 |
| [罕見字代碼上傳](功能模組/罕見字代碼上傳/) | 預留罕見字代碼上傳流程。 |
| [系統代碼更新回報](功能模組/系統代碼更新回報/) | 預留系統代碼更新與回報流程。 |
