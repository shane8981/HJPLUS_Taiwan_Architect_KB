# 新北市建造執照申請文件清單

本文件整理新北市轄區建造執照申請時常用的書類文件清單。新店區屬新北市轄區，建照書類文件原則上適用新北市建造執照申請規則；實際是否加掛山坡地、水土保持、都市設計審議、環境影響評估、捷運開挖安全、地質敏感等文件，仍需依基地條件判斷。

資料來源基準：新北市政府工務局「建造執照申請案件文件一覽表」108.10.16 版。送件前請再以新北市政府工務局網站及最新表單公告為準。

## 法令適用日與抽換限制

依提供資料整理，官方備註中列為法令適用日文件者包括：

- 6. 建造執照申請書
- 10. 地號表
- 18. 土地使用權同意書
- 19. 土地登記第一類謄本
- 20. 地籍圖謄本

上述文件原則上不得任意抽換。若建築基地面積異動，通常需重新檢附 6、10、14、15、18、19、20、21、22 等文件；若僅涉及地號變更且未增加基地面積，通常需重新檢附 10、18、19、20 等文件。

## 新店區常見加掛條件

新店區不少基地可能涉及坡地、地質敏感、水土保持、捷運沿線、都市設計審議或環評門檻。送件前建議先完成下列基地條件篩選。

| 檢核條件 | 可能增加文件 |
|---|---|
| 是否屬山坡地或坡審範圍 | 山坡地建築管理與技術規範檢核表、坡審報告書、地質／大地相關報告 |
| 是否涉及水土保持 | 水土保持計畫或簡易水保相關核定文件 |
| 是否屬地質敏感區 | 地質敏感查詢結果、地質安全評估報告 |
| 是否屬土壤液化潛勢區 | 土壤液化潛勢區自行檢核表 |
| 是否須都市設計審議 | 都審報告書、核定版都審文件 |
| 是否鄰近捷運或大型開挖影響範圍 | 捷運開挖安全影響評估或相關外審報告 |

## 無紙化送件提醒

新北市建築執照審查已導入無紙化流程。建造執照、雜項執照、拆除執照、建照報備屬實施類型；建築申請書表仍需檢附紙本用印，並同步上傳 PDF。綠建築報告書、結構計算書、鑽探報告書及外審報告書，依規定上傳 PDF 或掃描檔。圖說類則以建築執照相關圖說、結構圖說 PDF 上傳為主，並以彩色圖為原則。

首次掛號前，應至建築執照申請書表系統填寫申請書、上傳書圖、完成資料上傳，取得一碼通後列印無紙審照相關報表，再檢附檢核表所列紙本書件送新北市政府工務局掛號。

## 使用建議

- 先依「基本申請文件清單」建立送件目錄。
- 再依基地條件補入山坡地、水保、地質敏感、都審、環評、捷運開挖安全等條件式文件。
- 送件前以新北市政府工務局最新公告、建築執照申請書表電子化系統、以及最新表單版本覆核。
- 若案件位於新店區山坡地或地質敏感範圍，應提早確認坡審、水保、地質安全評估與外審時程。
