# 文化資產保存法

本分類收錄與台灣文化資產保存法相關的知識條目，主要協助從業人員在設計規劃階段，判斷基地周邊或基地內可能涉及的文化資產項目，並依相關法規提出應對方案。

## 知識條目

| 技能名稱 | 說明 | 類別 |
|---------|------|------|
| [文化資產項目](文化資產項目/) | 文化資產保存法中定義的各類文化資產項目（有形、無形及保存技術），並說明其法規限制。 | C 類台灣法規 |
| [修復再利用因應計畫](修復再利用因應計畫/) | 文化資產修復與再利用時，提送因應計畫以排除都市計畫法、建築法、消防法相關規定之法規機制。 | C 類台灣法規 |
| [修復工程程序](修復工程程序/) | 依古蹟修復及再利用辦法，解說修復計畫的四個法定階段：調查研究、修復設計、監造、工作報告書。 | C 類台灣法規 |
| [文化資產獎勵與補償](文化資產獎勵與補償/) | 文化資產所有權人之財務獎勵機制，含容積移轉（TDR）、免徵房屋稅及地價稅、修復補助申請。 | C 類台灣法規 |
