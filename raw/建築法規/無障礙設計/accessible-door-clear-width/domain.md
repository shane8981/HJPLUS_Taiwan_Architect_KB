# 無障礙出入口門淨寬實務陷阱

> 規範寫的「門框間距離 ≥ 90cm」是**開口的標稱值**；但人實際能通過的是**門開啟後的有效淨寬**。
> 兩者之間被門扇厚度、五金與裝修貼面吃掉——這格才是本卡片的價值。

## 使用情境

- 規劃無障礙通路上的出入口、決定門開口尺寸時
- 圖面標註門寬、要決定是否替門扇佔用與貼面預留時
- 無障礙竣工查驗前自主檢查、想避免完工才爆時

## 核心觀念：標稱開口 ≠ 有效通行淨寬

規範條文量的是「**門框間之距離**」（鉸鏈門 ≥ 90cm）

門檻是「**大於等於**」，所以開口畫剛好 90cm，貼完面、裝上門扇後有效淨寬必定不足。

## 陷阱卡片：無障礙門淨寬（有效通行淨寬陷阱）

| 欄位 | 內容 |
|---|---|
| **法規依據** | 建築物無障礙設施設計規範 **§205.2.3（出入口）**；不得使用旋轉門、彈簧門（§205.4.1） |
| **法規門檻** | 鉸鏈門：**門框間之距離 ≥ 90cm**；橫向拉門、折疊門：**開啟後之淨寬度 ≥ 80cm**。地面須平順、不得設門檻 |
| **圖面對策** | 開口常畫到 **100cm**，預留門扇佔用與貼面，確保完工後有效淨寬 ≥ 90cm |
| **失敗後果** | 有效淨寬 < 90cm → 無障礙設施查驗不通過 → 卡使照、無法取得無障礙合格認定 |
| **後果時點** | 裝門、貼面後才顯現，事後難補救（須改門扇形式或重做門框） |
| **連動法規／作業** | 無障礙通路全線淨寬、室內裝修、使照竣工查驗 |
| **嚴重度** | 🔴 高（直接卡使照，且屬完工後才爆的後段風險） |

### 一句話記憶

> 90cm 要在「門開好、貼完面」之後量得到；圖上就畫 100cm，把門扇和貼面的厚度先讓出來。

## 設計檢核重點

- [ ] 鉸鏈門開口是否預留門扇佔用與貼面（門框間建議 ≥ 95–100cm，確保有效淨寬 ≥ 90cm）？
- [ ] 是否誤用旋轉門、彈簧門？（§205.4.1 禁止）
- [ ] 門檻是否做到平順無高差？
- [ ] 橫向拉門、折疊門開啟後淨寬是否 ≥ 80cm？
- [ ] 門把是否留 4–6cm 防夾手空間（§205.4.3）？

## 條文原文（§205.2.3，節錄）

> 「門扇打開時，地面應平順不得設置門檻，且門框間之距離不得小於九十公分。橫向拉門、折疊門開啟後之淨寬度不得小於八十公分。」
